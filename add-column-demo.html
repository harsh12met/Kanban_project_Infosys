Add New Column Feature Demo

About This Feature

The Kanban Board now allows you to add custom columns to better organize your tasks. Each column has a "+" button that lets you add a new column immediately after it.

This demo walks you through the process of using this new feature.

1 Locate the "+" Button

Each column has a "+" button in its top-right corner, next to the task count.
